# 📈 Real-Time Stock Market Dashboard

## 📌 Project Overview

The Real-Time Stock Market Dashboard is a web-based application that allows users to track and visualize live stock market data. Users can enter a stock symbol and view dynamic charts, company information, and key financial indicators in real time.

---

## 🚀 Features

* 📊 Live stock price tracking
* 📉 Interactive price charts using Plotly
* 📦 Volume analysis
* 📈 Moving average (trend analysis)
* 🏢 Company details (name, sector, market cap)
* ⏱️ Multiple time period selection (1D, 5D, 1M, etc.)
* 🎯 User-friendly interface built with Streamlit

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **Streamlit** – Web application framework
* **Pandas** – Data handling and analysis
* **Plotly** – Interactive data visualization
* **yFinance API** – Real-time stock data

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Neethusree111204/Stockmarket.git
cd stock-dashboard
```

2. Install dependencies:

```bash
pip install streamlit yfinance pandas plotly
```

3. Run the application:

```bash
streamlit run app.py
```

4. Open in browser:

```
http://localhost:8501
```

---

## 📊 How to Use

* Enter a valid stock symbol (e.g., AAPL, TSLA, MSFT)
* Select the desired time period
* View real-time graphs and stock insights
